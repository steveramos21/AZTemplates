# FirstVMTemplate
This template deploys the following:
Operating System : WindowsServer 2012-R2-Datacenter
VMSize : Standard_B2ms
OSDisk : Standard_LRS
VirtualNetwork : 10.0.0.0/16
Subnet : 10.0.0.0/24
